BathyScaphe の新機能

バージョン 2.4.3 以降の BathyScaphe には、２ちゃんねるのブラウズをさらに快適にするための、いくつかの新機能が含まれています。

掲示板リストの同期先を変更する (バージョン 2.4.4)

掲示板リストの同期に使用する bbsmenu の URL を自由に設定できるようになりました。また、「掲示板を追加」画面から設定や同期がすぐに行えるようになりました。

  • 「詳細」を設定する
  • 掲示板を追加する

OS X Yosemite への対応

OS X Yosemite の新しい外観にマッチするよう、各部のデザインを見直しました。また、互換性の問題も修正されています。

掲示板の名前を後から変更する

ありそうで無かった機能がついに登場です。掲示板の名前が本当に変更された場合や、あるいは独自の名前を付けたい場合などに、既にあるログやデータベースの整合性を保つ安全な方法で変更を行えるようになりました。

  • 掲示板の名前を変更する

２ちゃんねるの新機能に対応

キャップ付きや BE 2.0 による名前欄の色付き表示に対応します。また、ID の末尾の文字種別の増加に対応し、迷惑レスの条件として使用できます。

まちBBSのホスト表示が復活

まちBBSの offlaw.cgi のアップデートに対応し、まちBBSのレスで再びホストの情報が取得・表示できるようになりました。
