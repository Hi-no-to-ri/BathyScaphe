リンクをプレビューする

BathyScaphe では、一部の外部リンクを Web ブラウザを使用せずにプレビューすることができます。内蔵のプレビュー機能は、主に画像ファイルのリンクをプレビューすることができます。

  • リンクをプレビューするには、単にそのリンクをクリックします。
  • プレビュー機能を使用したくない場合は、「リンク」環境設定で設定します。もっと詳しく知りたいときは、関連項目のリンクをクリックしてください。

リンクがプレビュー可能な場合は、小さなプレビュー用のウインドウが開き、そこにリンクの内容が表示されます。そうでなければ、かわりに Web ブラウザでそのリンクが開かれます。

ヒント：プレビュー機能は「プラグイン」形式になっているため、他の開発者が作成したプラグインを別途インストールすることで、他の種類のリンクもプレビューしたり、あるいはより高機能なプレビューを行うことができるようになります。
