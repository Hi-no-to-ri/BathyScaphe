特定の ID を抽出したいのに、うまくできない

  • 特定の ID の発言をすべて見たい場合は、「検索」パネルを開かなくても、ID 文字列（ID 欄および本文中の「ID:」で始まる文字列）をクリックしたまま少し待つだけで抽出できます。
    注意：ID が「???」で始まっている場合、または、その ID の発言がクリックした ID 欄のレスだけしか無い場合は、クリックしても抽出されません。
  • クリックしたまましばらく待っても抽出できない場合は、「詳細」環境設定の「ID や逆参照をポップアップする方法」の設定が変更されている可能性があります。
    • 「マウスオーバー」が選択されていると、クリックしなくても表示されます。
    • 「クリック」が選択されていると、クリックするだけで（待たずに）表示されます。
    • 「長押し」が選択されていると、クリックしたまま少し待つことで表示されます。

関連項目

  • 「詳細」を設定する
  • 内容を検索する
