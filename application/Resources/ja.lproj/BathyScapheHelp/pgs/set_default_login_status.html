浪人、Be、および公式 p2 の使用をあらかじめ設定する

浪人、Be（BE 2.0）、および公式 p2 を使用した書き込みを頻繁に行う場合、書き込みウインドウを開いた時点でこれらのアカウントを使用する状態にしておくかどうかを設定することができます。

これらの設定は、「すべての掲示板」または「特定の掲示板」ごとに設定できます。

  • すべての掲示板で浪人、Be、または公式 p2 の使用／未使用をあらかじめ設定するには、「アカウント」環境設定でそれぞれのアカウントを左のリストから選択します。次に、右に表示される「浪人（または Be）ログインして書き込む」または「公式 p2 を使用して書き込む」チェックボックスをクリックします。
  • 掲示板ごとに浪人、Be、または公式 p2 の使用／未使用をあらかじめ設定するには、「掲示板」＞「掲示板オプションを表示」と選択します。次に「書き込み」をクリックし、それぞれのアカウントのチェックボックスをクリックします。
    参考：掲示板ごとの設定は、すべての掲示板に対する設定よりも優先されます。

掲示板によっては、BathyScaphe またはアカウントサービスそのものが、そのアカウントを使用した書き込みをサポートしていなかったり、逆に使用が必須であることがあります。このような掲示板では、「掲示板オプション」の「書き込み」で、そのアカウントのチェックボックスが淡色表示になり、クリックできなくなります。これは異常ではありません。

このように設定した内容は、書き込みウインドウを開いたときに「あらかじめ設定されている」だけであることに注意してください。書き込みウインドウのツールバーにある「浪人」「Be」および「p2」の各ボタンをクリックすれば、浪人、Be、および公式 p2 の使用／未使用はいつでも変更できます。もし、ツールバーにこれらのボタンが見つからない場合は、ツールバーをカスタマイズして追加することができます。
