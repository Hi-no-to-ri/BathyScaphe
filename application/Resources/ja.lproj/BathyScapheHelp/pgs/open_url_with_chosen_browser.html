Safari 以外の Web ブラウザを常用しているのに、外部リンクが Safari で開いてしまう

Safari 以外の Web ブラウザを常用しているのに外部リンクが Safari で開いてしまう場合、以下の点を確認する必要があります。

  • OS のデフォルト Web ブラウザが Safari のままになっていないかをまず確認します。デフォルト Web ブラウザの設定を確認、変更する方法については、以下のページを参照してください。
    外部リンクを開く Web ブラウザを変更する
  • デフォルト Web ブラウザを Safari 以外のブラウザに設定しているにもかかわらず Safari で開いてしまう場合は、どのような方法でその外部リンクを開こうとしたかを確認してください。
    • たとえば、書き込みウインドウなどのコンテキストメニューに「URL を開く」という項目が表示される場合がありますが、これは「Safari」が表示しているメニューです。このため、設定とは関係なく常に「Safari」が使用されます。これは BathyScaphe の問題ではありません。

上記の問題を避けるには、次の設定を行います。

  1. 「キーボード」システム環境設定を開きます。
  2. 「ショートカット」タブを選択します。
  3. 左のリストで「サービス」を選択します。
  4. 右のリストで「URL を開く」という項目を探し、チェックボックスをクリックしてチェックマークを外します。
    これで、Safari で開く操作ができなくなります。
  5. さらに、Web ブラウザによっては右のリストに「URL を（ブラウザ名）で開く」といった項目が用意されている場合があります。その場合はその項目にチェックマークを付けることで、その Web ブラウザでリンクを開くことができるようになるでしょう。
