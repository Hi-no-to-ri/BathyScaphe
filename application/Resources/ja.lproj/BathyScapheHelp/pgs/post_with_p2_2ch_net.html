公式 p2 を使用して書き込む

公式 p2（http://p2.2ch.sc/）のサービスを BathyScaphe から利用して、2ch.sc 掲示板に書き込むことができます。

お使いのネットワーク環境が 2ch.sc に規制されているなどの原因で書き込みができない場合でも、公式 p2 を使用して書き込みを行える場合があります。公式 p2 は通常 Web ブラウザから利用しますが、BathyScaphe をすでにお使いであれば、使い慣れたインタフェースからそのまま公式 p2 を利用した書き込みを行えるので便利です。

重要：公式 p2 から書き込みを行うには、あらかじめ Web ブラウザから公式 p2 にアクセスして、書き込み機能を有効化しておく必要があります。また、書き込み機能の有効化にはモリタポと呼ばれるオンライン通貨が必要です。

公式 p2 を使用して書き込むには：

  1. あらかじめ Web ブラウザから公式 p2 にログインして、書き込み機能を有効化しておきます。
  2. BathyScaphe で「アカウント」環境設定を開き、左のリストで「公式 p2」をクリックします。次に、メールアドレスとパスワードを入力しておきます。
    • パスワードは空欄にしておくこともできます。空欄にしておくと、実際に公式 p2 を使用して書き込みを行う際に BathyScaphe が都度パスワードの入力を求めてきます。
  3. スレッドに書き込む際に、ツールバーの「p2」ボタンをクリックして、「p2」アイコンを青く点灯した状態にします。
    • ツールバーに「p2」ボタンが見つからない場合は、ツールバーをカスタマイズして追加します。
    • すべてのスレッド、または特定の掲示板ごとに、あらかじめ「p2」アイコンが青く点灯した状態にしておくこともできます。
  4. 書き込みを実行します。

公式 p2 を使用して書き込みを行う際の制約事項や手順についてより詳しくは、BathyScapheWiki を参照してください。
