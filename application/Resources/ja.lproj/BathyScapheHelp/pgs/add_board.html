掲示板を追加する

掲示板リストに掲示板を追加するには、以下のようにします（スマート掲示板を追加したい場合は、かわりに「スマート掲示板を作成する」を参照してください）。

  1. 「掲示板」＞「掲示板を追加...」と選ぶか、または掲示板リストの左下にある追加（＋）ボタンをクリックします。
  2. リストからカテゴリや掲示板を選択して「追加」をクリックします。検索フィールドを利用すると、すばやくリストから探すことができます。

最近新設された掲示板や、一部の２ちゃんねる以外の掲示板は、リストに掲載されていない場合があります。このような場合は、まずリストを最新の状態に更新してみます。また、より多くの掲示板が掲載されたリストを使用してみることもできます。それでも見つからない場合は、直接掲示板の名前と URL を入力して追加できます。

リストを最新の状態に更新するには：

  1. 「掲示板」メニューから「掲示板リストを同期」と選択します。
    すでに「掲示板を追加」画面を開いているときは、かわりにリストの左下にある更新ボタン（反時計回りの矢印のボタン）をクリックしても同じです。

リストに掲示板が見つからない場合に、ほかのリストに変更するには：

  1. 「BathyScaphe」メニュー（アプリケーションメニュー）から、「環境設定...」を選択し、次に「詳細」をクリックします。
    すでに「掲示板を追加」画面を開いているときは、かわりにリストの左下にある歯車のボタンをクリックしても「詳細」環境設定を開くことができます。
  2. 「掲示板リストの同期先：」からほかの URL を選択するか、bbsmenu を提供しているほかの Web ページの URL を直接入力します。
  3. すぐに同期するかどうかを尋ねるダイアログが表示されたら、「同期」をクリックします。

リストに掲示板が見つからない場合に、手動で掲示板を追加するには：

  1. 「掲示板を追加」画面の下部に直接掲示板の名前と URL を入力します。
  2. 「追加」をクリックします。

ヒント

  • したらばや、小規模な互換掲示板の場合は、リストに載っていないことが多いため、直接入力する必要があるでしょう。
  • 掲示板の URL は「http://」で始まり、「/」で終わっていなければなりません。末尾に「index.html」などがついている場合は、それを削ったものを入力します。
